    第一章電腦概論

    維讀記憶體(ROM)，只能讀取，不能讀出。

    ROM被儲存在一個非揮發性晶片上，也就是說，即使在關機之後記憶的內容仍可以被儲存，所以這種記憶體多用來儲存特定功能的程式，如韌體。ROM儲存用來啟動電腦的程式（如BIOS）



    RAM 是揮發性記憶體，代表資訊僅暫時儲存於模組內，並會在重新開機或關機後清除。由於資訊以電力儲存於電晶體內，當電流中斷後，資料隨即消失。每次您發出要求時，它就會透過電腦的儲存磁碟或網路，來擷取所需的檔案或資訊。由於資料儲存在 RAM 中，故每次您在程式或網頁間切換時，資料為立即可用狀態。電腦關機後，將清空記憶體直到程序再次啟動。揮發性記憶體可由使用者輕鬆進行更換、升級、或擴充。



    可程式唯讀記憶體（Programmable ROM，PROM）其內部有行列式的鎔絲，可依使用者（廠商）的需要，利用電流將其燒斷，以寫入所需的資料及程式，鎔絲一經燒斷便無法再恢復，亦即資料無法再更改。

    EEPROM 電子抹除式可複寫唯讀記憶體（Electrically Erasable Programmable Read Only Memory，EEPROM）之運作原理類似EPROM，但是抹除的方式是使用高電場(紫外線光)來完成，因此不需要透明窗。

    輸出設備和輸入設備合稱為電腦的週邊設備(peripheral equipment)，大致分為：輸入設備、輸出設備及輸入/輸出設備三種

    輸入媒體:二維碼、金融卡

    輸出媒體:用來記錄計算機輸出的媒體，例如紙張、紙帶、磁帶、磁碟等。

    資料傳輸速度單位 : BPS

    CPU I/O負責輸入輸出，ALU負責算術運算與邏輯運算。



    控制單元（Control Unit），有時為CPU一部分，有時安裝於CPU外部，負責指揮CPU工作。通過該裝置的運行來控制其他裝置的活動，也被視作有限狀態自動機的一種。

    CPU的控制單元曾經只被當成暫時性的通路，其設計十分困難。 目前的控制單元多採用被包含於儲存控制器的微程式加工製造。工作時由微型定序器選定微程式代碼，其各位元組即負責控制計算機的各個部分。諸如暫存器，算術邏輯單元，指令暫存器，總線，甚至晶片外部的輸入輸出均在其掌控之中。在當前的計算機中，各子系統分別擁有隸屬於控制單元的控制器，由這些控制器監督各子系統工作，負責從主記憶體讀取並解釋指令。

    雙工（duplex）， 指二台通訊裝置之間，允許有雙向的資料傳輸，可細分為半雙工和全雙工。

    半雙工無法雙向傳輸，無線對講機就是使用半雙工系統。

    一般的電話、手機就是全雙工的系統，因為在講話時同時也可以聽到對方的聲音。

    單工:計算機和印表機之間的通信是單工模式，因為只有計算機向印表機傳輸數據，而沒有相反方向的數據傳輸。還有在某些通信信道中，如單工無線發送等。

    輔助記憶體又稱外部記憶體。存放在RAM中的資料會因關機而消失，如果在關機之前將RAM中的資料存放於輔助記憶體之中，就可於日後隨時取用。

    快閃記憶體（英語：flash memory），是一種電子清除式可程式唯讀記憶體的形式，允許在操作中被多次擦或寫的記憶體。這種科技主要用於一般性資料儲存，以及在電腦與其他數位產品間交換傳輸資料，如記憶卡與隨身碟。快閃記憶體是一種特殊的、以大區塊抹寫的EEPROM。早期的快閃記憶體進行一次抹除，就會清除掉整顆晶片上的資料。

    快閃記憶體的成本遠較可以位元組為單位寫入的EEPROM來的低，也因此成為非揮發性固態儲存最重要也最廣為採納的技術。像是PDA、固態硬碟、筆記型電腦、數位隨身聽、數位相機與手機上均可見到快閃記憶體。此外，快閃記憶體在遊戲主機上的採用也已經普及，藉以取代儲存遊戲資料用的EEPROM或帶有電池的SRAM。

    雙絞線（英語：Twisted pair）是由兩條外面被覆塑膠類絕緣材料、內 含銅纜線，互相絕緣的雙線互相纏繞（一般以順時針纏繞），絞合成螺旋狀的一種電纜線。雙絞線可減少傳送中訊號的衰減、減少串擾（英語：crosstalk）及雜訊（英語：Noise）、並改善了對外部電磁干擾的抑制能力。(ADSL)
