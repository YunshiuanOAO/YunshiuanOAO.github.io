    nlogn-x 找因數

    可以利用建表將\(1~x\)每個到\(X\)的倍數個數存進陣列中，每次詢問時可以直接使用。 

    for(int i=1;i<=SIZE;i++){
   for(int j=i;j<=SIZE;j+=i)
      c[j]++;
   }
}
